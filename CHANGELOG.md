# Changelog

All notable changes to Valgaron World Codex will be documented here.

The project follows semantic versioning once public releases begin.

## 0.0.0 - Unreleased

### Added

- Local-first World Codex for project/universe workspaces, in-fiction
  worlds/planets, codex entries, relationships, timelines, custom entry types,
  import/export, recovery snapshots, diagnostics, PWA metadata, and GitHub Pages
  deployment preparation.
- In-app Help route, user guide, support guidance, runtime recovery QA, static
  hosting docs, versioning policy, and manual release checklist.
- Schema 4 entry cover/gallery images from HTTPS sources or verified local JPEG,
  PNG, WebP, and GIF uploads, with accessible metadata and web/mobile local
  binary persistence.
- Active-workspace and full-document ZIP export/import containing exact JSON and
  reachable uploaded images, with integrity verification and safe replacement.
- Lossless schema 3 migration, snapshot-aware asset cleanup, content-free image
  diagnostics counts, and native picker/filesystem/share-sheet parity.

### Notes

- This is still a pre-release local-only prototype. There are no accounts, cloud
  sync, collaboration, telemetry, hosted services, payments, or localization.
- JSON remains a URI/metadata portability path; ZIP is required to restore
  uploaded image bytes.
